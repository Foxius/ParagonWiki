# Некроплесень

Плесень - это предмет в Мании, используемый для превращения растений в трупы.



## Получение

Можно купить у бессонницы за лунные камни, а также с вероятностью 15% выпадает из Труповзрывов. Плесень также можно найти в сундуках в могиле Мавзолея.

## Применение

При использовании на некоторых растениях плесень превращает их в свою трупную версию.

|               Растение               |     Трупорастение     |
| :----------------------------------: | :-------------------: |
|             Высокие цветы            |       Трупоцвет       |
|              Тыква, дыня             |     Трупные семена    |
|                Кактус                |     Трупный тунец     |
| Маленький или большой капельный лист | Капсула трупокопателя |
|                Аллиум                | Стебель трупного Лука |
|       Сахарный тростник, бамбук      |      Трупофлейта      |

При успешном использовании плесени другие близлежащие блоки превращаются в песок душ, почву души и каменистую почву.
